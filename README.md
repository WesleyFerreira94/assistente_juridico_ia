# 🎓 Projeto de Extensão: Assistente Jurídico com Inteligência Artificial

Este repositório contém o código-fonte de um protótipo de Assistente Jurídico desenvolvido como parte das atividades práticas do curso de Análise e Desenvolvimento de Sistemas. 
O objetivo do projeto foi aplicar conceitos de lógica de programação, manipulação de banco de dados relacional e integração de inteligência artificial de forma local para resolver um problema do mundo real.

## 📝 Descrição do Projeto

O sistema simula o funcionamento de um assistente inteligente voltado para a rotina de um escritório de advocacia. 
Ele oferece uma interface gráfica para que o usuário possa interagir com um modelo lógico de tomada de decisões e organização de dados contratuais e processuais.

Como diferencial acadêmico, o projeto foi arquitetado para funcionar de forma **100% local**, sem a necessidade de chaves de API pagas ou conexão com servidores externos, 
garantindo privacidade e custo zero de execução.

## 🛠️ Tecnologias e Conceitos Aplicados

* **Linguagem:** Python (estruturas de dados, lógica e funções).
* **Interface Gráfica (GUI):** Tkinter (desenvolvimento de telas, botões e gerenciamento de eventos).
* **Banco de Dados:** SQLite3 (criação de tabelas, persistência de dados local e consultas SQL).
* **Inteligência Artificial:** Modelagem e lógica de processamento local de linguagem natural.

## 📁 Estrutura de Arquivos

* `main.py`: Arquivo principal que inicializa a interface gráfica do sistema.
* `database.py`: Scripts de criação e conexão com o banco de dados SQLite3.
* `assistente.py`: Arquivo responsável pela lógica de inteligência e processamento das respostas.
*(Nota: Ajuste os nomes dos arquivos acima caso os seus arquivos tenham nomes diferentes!)*

## 🚀 Como Rodar o Protótipo

### Pré-requisitos
Ter o **Python 3.x** instalado no computador.

### Passo a Passo no Terminal

1. Clone este repositório para a sua máquina:
```bash
   git clone [https://github.com/SEU-USUARIO/assistente-juridico-ia.git](https://github.com/SEU-USUARIO/assistente-juridico-ia.git)
   
2. Acesse a pasta do projeto:

Bash
   cd assistente-juridico-ia

3. Execute o arquivo principal para abrir a interface:

Bash
   python main.py
